# Paddle Score Game

## Overview
This project is for the CISC191 Final. It utilizes the various modules from 1 to 7 in order to create a mockup twist of Pong.

### Instructions for running the project:

- Run `mvn javafx:run` in Maven

### Instructions for running the tests:

- Run `mvn test` in Maven from the root folder or run individually in `src/test/java/edu/sdccd/cisc191/`
